关于寒假的计划

2021-01-22
いろいろ
约 1858 字 预计阅读 4 分钟

最近太忙了，婆婆重病，offer，开题答辩……导致好久没更新了。

由于已经拿到offer了，寒假的计划要做相应的改动，主要关注的是，日语（希望能稳固达到N3水平），设计模式和计算机基础，计网和数据结构，算法，Spring。

书单如下——

  • 大家的日语 初级1，2（这两本应该是N3水平…或许等7月份有能力去考N2呢？なんちゃって〜）
  • 图解设计模式/设计模式的艺术:软件开发人员内功修炼之道
  • CSAPP
  • 图解HTTP
  • 算法图解/算法第四版
  • github入门与实践
  • SQL必知必会
  • Spring4.x….

书单看起来多，但是其中好几本，比如git，SQL之类的都应该是比较容易看完的……总之无论如何，加油。

想达到的目标如下，按重要性降序排列——

  1. 日语N3（不止要考级，还要有单词积累，口语能力，写作能力…毕竟学日语的目的是为了将来去日本工作，而非单纯为了个什么所谓的“等级证书”）
  2. Spring等框架
  3. 巩固计算机基础（CSAPP）
  4. 深入学习算法
  5. 巩固离散数学（见此）
  6. git，SQL
  7. 计网的基础（主要是HTTP）
  8. 学习emacs的使用

以及画画……画画是一辈子的兴趣。

对函数式编程，面向语言编程，逆向工程，编译原理，网络安全等感兴趣，但是这不是目前应当关注的目标。

（下面的内容编写于20年12月27日）

反省

这一学期过得比较尴尬。学期刚开始时，打算坚持画画和学习马克思，结果到十月中旬，其他人都差不多找到工作的时候，突然（通过导员）意识到，啊，原来这一学期要tm求职（并且上个暑假应当去参加一些实习的，这些机会我全都遗漏过去了！可见搜集信息的重要性）！可是已经晚啦，并且由于基础比较差，成绩也比较差的缘故（拿得出手的只有手里的两个奖），现在还没有找到工作。

这两个月真是不好受，在希望与失望中反复横跳。但是无论如何，我明确了自己要走的道路（也积累了一些面试经验），而且也学了很多JavaSE基础和算法，有了一点底子。于是，这个寒假我也要厉兵秣马，积极学习来准备春招了。

计划

这个寒假基本上可以算是从12月30日开始，到明年2月28日结束（当然，不考虑疫情的情况下）。

寒假有两个方面需要关心——就业和论文。

关于就业，主要有三个方面是需要重点涉及的——

  • JavaEE（包括JavaWeb，Spring Boot， myBatis等）
  • 数据结构和算法
  • 设计模式（和JavaSE，既要巩固，也要继续深入，不仅要知道它有啥，怎么用，还要知道它是怎么做的，为什么那么做，不那么做会怎样）

这几个方面是需要了解的——

  • JVM
  • 计算机网络（特别是HTTP）

同时，也应当补充一些计算机网络的知识，

关于论文，在研究相关的算法的基础上，还得学习一些前端的知识，如nodejs，js（或ts），ajax，以及axios库，echarts库和layUI库的使用。其后端使用Spring和sql，和就业方面是重叠的。

总之，这个寒假，我要让自己拥有如下能力——

  • 熟悉JavaEE，熟悉Spring，Spring MVC，Spring Boot，myBatis，有项目经验（！）
  • 熟悉常用数据结构和算法（比如手撕红黑树？hh）
  • 精通SQL语句，熟悉mySQL和Oracle等数据库（估计还需要了解一点数据库设计，不过具体要学啥我还不知道）
  • 熟悉Java语言，熟悉Java的容器类，阅读过相关源码
  • 熟悉git等版本控制工具
  • 熟悉http
  • 了解JVM，了解JVM调优
  • 了解html，js，css，jquery（其实要当后端程序员，学点前端也是必要的）
  • 分布式的一些玩意究竟需不需要去了解？如redis，zookeeper之类的？如果学有余力的话吧……

书单

Java SE & JVM

  • On Java 8，此书不止关系到Java，还涉及到面向对象，设计模式等内容，并且比较基础，所以比较值得看（但是翻译着实有点不好）
  • java核心技术 卷1卷2
  • 实战Java虚拟机 / 深入理解JVM虚拟机

Java EE & sql & etc…

  • Spring4.x企业开发实践（还要一些JavaWeb基础……如果此书里讲解不够的话）
  • 图解HTTP
  • sql必知必会（此书只是用来学习SQL语句的，缺少数据库设计上的东西）
  • mybatis
  • 一些前端的东西……
  • git入门与实践

设计模式

  • 图解设计模式（结城浩）
  • 大话设计模式

数据结构和算法

  • 剑指Offer
  • 算法图解
  • 算法第四版 (此书有闲暇再看)

需要在看书的基础上多实践，也要重新拾起leetcode

总结

这次不定什么schedule，实践证明我从来都完不成预订计划。但是这次或许会真的不一样？因为和之前所有的实践不同，这一次我真真切切面临着经济压力……我应该积极拥抱它，它会让我作出切实的改变，从一个满脑子幻想的学生变成真正的…社会中的一员。
