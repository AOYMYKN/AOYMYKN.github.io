关于寒假的计划

2020-12-27
いろいろ
约 1243 字 预计阅读 3 分钟

反省

这一学期过得比较尴尬。学期刚开始时，打算坚持画画和学习马克思，结果到十月中旬，其他人都差不多找到工作的时候，突然（通过导员）意识到，啊，原来这一学期要tm求职（并且上个暑假应当去参加一些实习的，这些机会我全都遗漏过去了！可见搜集信息的重要性）！可是已经晚啦，并且由于基础比较差，成绩也比较差的缘故（拿得出手的只有手里的两个奖），现在还没有找到工作。

这两个月真是不好受，在希望与失望中反复横跳。但是无论如何，我明确了自己要走的道路（也积累了一些面试经验），而且也学了很多JavaSE基础和算法，有了一点底子。于是，这个寒假我也要厉兵秣马，积极学习来准备春招了。

计划

这个寒假基本上可以算是从12月30日开始，到明年2月28日结束（当然，不考虑疫情的情况下），算它60天吧。

寒假有两个方面需要关心——就业和论文。

关于就业，主要有三个方面是需要重点涉及的——

  • JavaEE（包括JavaWeb，Spring Boot， myBatis等）
  • 数据结构和算法
  • 设计模式（和JavaSE，既要巩固，也要继续深入，不仅要知道它有啥，怎么用，还要知道它是怎么做的，为什么那么做，不那么做会怎样）

这几个方面是需要了解的——

  • JVM
  • 计算机网络（特别是HTTP）

同时，也应当补充一些计算机网络的知识，

关于论文，在研究相关的算法的基础上，还得学习一些前端的知识，如nodejs，js（或ts），ajax，以及axios库，echarts库和layUI库的使用。其后端使用Spring和sql，和就业方面是重叠的。

总之，这个寒假，我要让自己拥有如下能力——

  • 熟悉JavaEE，熟悉Spring，Spring MVC，Spring Boot，myBatis，有项目经验（！）
  • 熟悉常用数据结构和算法（比如手撕红黑树？hh）
  • 精通SQL语句，熟悉mySQL和Oracle等数据库（估计还需要了解一点数据库设计，不过具体要学啥我还不知道）
  • 熟悉Java语言，熟悉Java的容器类，阅读过相关源码
  • 了解JVM，了解JVM调优
  • 了解html，js，css，jquery（其实要当后端程序员，学点前端也是必要的）
  • 了解http

  • On Java 8，此书不止关系到Java，还涉及到面向对象，设计模式等内容，并且比较基础，所以比较值得看（但是翻译着实有点不好）
  • java核心技术 卷1卷2
  • 实战Java虚拟机 / 深入理解JVM虚拟机

  • 一些前端的东西……

总结

这次不定什么schedule，实践证明我从来都完不成预订计划。但是这次或许会真的不一样？因为和之前所有的实践不同，这一次我真真切切面临着经济压力……我应该积极拥抱它，它会让我作出切实的改变，从一个满脑子幻想的学生变成真正的…社会中的一员。
